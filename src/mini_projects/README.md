# SecureBank ATM Simulator

## Overview

SecureBank ATM Simulator is a Java console-based banking application designed to simulate core ATM operations while strengthening foundational software engineering concepts.

The project focuses on:

* Java fundamentals
* Control flow
* Input validation
* Methods and clean coding
* State management
* Transaction handling
* Problem-solving and debugging

This project was built as part of an intensive Java fundamentals and enterprise engineering preparation journey.

---

# Features

## Authentication System

* Secure PIN login
* Maximum login attempts
* Account lock mechanism after repeated failures

## ATM Operations

* Check account balance
* Deposit funds
* Withdraw funds
* Transfer funds
* Change account PIN
* Exit system safely

## Transaction Management

* Mini statement generation
* Transaction history tracking
* Deposit and withdrawal logging

## Validation & Security

* Invalid menu handling
* Negative amount prevention
* Minimum balance enforcement
* PIN confirmation checks
* Recipient account validation
